# GitHub Copilot app

**Agent と進める開発を、ひとつのデスクトップ体験に。**

![](/assets/github-copilot-app/copilot-orb.png)

---

## 開発を動かす「コントロールセンター」

**Issue・Pull Request・CI・Agent Session** をひとつに。GitHub の文脈を保ったまま、指示からレビュー・マージまでつなぐ。

![](/assets/github-copilot-app/app-overview.png)

---

## Issue から Pull Request まで、ひとつながり

**人が判断し、Agent が実行。** 差分・テスト・CI を確認してチームへ届ける。

![](/assets/github-copilot-app/issue-to-pr.svg)

---

## 複数の仕事を、同時に

- Session ごとに**専用の branch と git worktree**
- バグ調査・機能実装・レビュー対応を、互いにぶつけず並列化

![](/assets/github-copilot-app/parallel-sessions.svg)

---

## Agent の任せ方を選べる

- タスクに合わせて **Interactive・Plan・Autopilot**
- モデルと reasoning effort も Session ごとに設定

![](/assets/github-copilot-app/session-modes.svg)

---

## チームの仕事に合わせられる

**ルール・専門知識・外部ツール・共有画面**を、いつもの仕事に組み込む。

![](/assets/github-copilot-app/make-it-yours.svg)

---
